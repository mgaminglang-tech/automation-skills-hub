## Featured Projects

### Solar Lead Automation CRM

**Problem:** Solar businesses need a simple way to collect, qualify, and manage leads.

**Solution:** Built a lead intake and CRM workflow using Tally, Airtable, n8n, Slack, and AI logic.

**Tools Used:** Tally, Airtable, n8n, Slack, AI Agent

**Business Result:** Centralized lead data, reduced manual checking, and improved follow-up visibility.

---

### Submission Tracking System

**Problem:** Teams manually track submissions, approvals, and escalations across multiple tools.

**Solution:** Built a workflow that captures intake data, stores records, sends review alerts, tracks status, and escalates overdue items.

**Tools Used:** Tally, Supabase, Slack, Retool, n8n

**Business Result:** Better pipeline visibility, fewer missed tasks, and cleaner internal operations.

---

### Renewal Management System

**Problem:** Businesses need a reliable way to track renewals and avoid missed deadlines.

**Solution:** Built a renewal tracking workflow with status monitoring, reminders, and dashboard visibility.

**Tools Used:** n8n, Supabase, dashboard UI, automation logs

**Business Result:** Improved renewal visibility and reduced manual follow-up.

---

### Facebook Auto Post System

**Problem:** Manual social posting takes time and creates inconsistent product promotion.

**Solution:** Built an automated product posting system using intake data, AI captions, image handling, and Facebook publishing.

**Tools Used:** n8n, Google Sheets, Facebook Graph API, AI captioning, Telegram

**Business Result:** Faster posting workflow, consistent captions, and better content operations.

---

### OLJ Job Scraper & Cover Letter Generator

**Problem:** Job applications take time to review and customize manually.

**Solution:** Built a Telegram-triggered workflow that extracts job details, analyzes fit, generates a tailored cover letter, and logs the result.

**Tools Used:** Telegram, n8n, HTTP Request, AI Agent, Google Sheets

**Business Result:** Faster job screening and more consistent customized applications.
